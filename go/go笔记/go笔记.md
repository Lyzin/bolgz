## 一、GO语言简介

### 1、Go包的官方地址

> 官网：https://golang.org/dl/
>
> 国内镜像站：https://golang.google.cn/dl

### 2、Go优势

> 交叉编译、编译快速
>
> 开发效率高
>
> 执行性能好
>
> 天生支持并发

### 3、学习记录

> 使用golang Idea编辑器或vs code，但是不要开启自动提示功能，使用省电模式，因为一开始就自己手写所有语法才能记得住

### 4、第一个Go程序

```go
package main

import "fmt"

func main() {
  fmt.Println("Hello world!!!")
}
```
